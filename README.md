# SQL-DATA-DIGGER-PROJECT# 

## Overview

Data Digger is a SQL-based database project designed to store, manage, and analyze business data. The project demonstrates the use of SQL queries, database design, data manipulation, and reporting techniques.

## Objectives

* Create and manage a relational database.
* Perform CRUD operations (Create, Read, Update, Delete).
* Implement primary and foreign key relationships.
* Generate reports using SQL queries.
* Practice joins, aggregate functions, subqueries, and database normalization.




## SQL Concepts Used

* CREATE DATABASE
* CREATE TABLE
* INSERT INTO
* SELECT
* WHERE
* ORDER BY
* GROUP BY
* HAVING
* Aggregate Functions (SUM, AVG, COUNT, MAX, MIN)
* Subqueries
* Foreign Keys


## Technologies Used

* MySQL
* SQL Workbench

## Conclusion

Data Digger provides hands-on experience with database creation, management, and data analysis using SQL. The project helps develop practical skills in database design and query optimization.
